- Python 3.13 이상을 사용합니다.
- 타입 힌트에는 `typing` 모듈 대신 built-in types를 사용합니다.
- Google Python Style Guide와 PEP 8을 따릅니다.
- 교육 자료를 수정할 때는 공식 문서와 좋은 예제를 적극적으로 참고하되, 수업 난이도와 맥락에 맞게 비판적으로 반영합니다.
